Introduction
============

Bienvenue dans le monde de Firebird $MAJOR.$MINOR.


Avertissements
==============

Firebird $MAJOR.$MINOR a été testé intensivement
et peut être utilisé en production. Toutefois, les
utilisateurs sont invités à suivre ces prescriptions
avant de déployer cette version en production. ie:

 o Lisez les notes de version et le fichier
   installation_lisezmoi.txt.

 o N'oubliez pas de sauvegarder vos données.

 o Il est recommandé de faire d'abords une
   désinstallation. Une désinstallation conserve vos
   fichier de log et de configuration.

 o Il est recommandé de faire vos propres tests
   avant tout déploiement en production.

Voir les notes de version (release notes) pour plus

Installation

doc de l'installation, dans l'Installation Guide ainsi

Rapporter des bugs
==================


Avant de rapporter un bug :

 o Vérifiez que vous comprenez comment Firebird fonctionne.
   Peut être que ce n'est pas un bug.

 o Quelqu'un a-t-il déjà rapporté ce bug? Regardez
   la liste de bugs rapportés ici:

    http://tracker.firebirdsql.org/browse/CORE

 o Si vous avez un doute pourquoi ne pas en parler à la liste
   Firebird-devel ? Vous pouvez vous inscrire ici:

    http://lists.sourceforge.net/lists/listinfo/firebird-devel

   et les messages de la liste sont accessible par newsgroup
   ici:

    news://news.atkin.com/
